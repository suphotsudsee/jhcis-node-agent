JHCIS Sync Desktop - คู่มือติดตั้งสำหรับเครื่องปลายทาง

ไฟล์สำคัญในแพ็กเกจนี้
- JHCISSyncDesktop_envonly.exe : โปรแกรมหน้าจอหลัก
- .env : ไฟล์ตั้งค่าการเชื่อมต่อฐานข้อมูลและ API
- .env.example : ตัวอย่างไฟล์ตั้งค่า
- scheduler_settings.json : ตั้งค่า Auto Sync ของโปรแกรม

วิธีติดตั้ง
1. แตก zip ไปไว้ในโฟลเดอร์ถาวร เช่น C:\JHCISSyncDesktop_envonly
2. เปิดไฟล์ .env แล้วแก้ค่าฐานข้อมูลและ API ให้ถูกต้อง
3. เปิด JHCISSyncDesktop_envonly.exe

วิธีใช้งาน
- ใช้ปุ่ม Send Data / Run Sync เพื่อส่งข้อมูลทันที
- ดูผลการทำงานได้ในหน้าจอโปรแกรมและในโฟลเดอร์ logs
- ถ้าต้องการให้โปรแกรมทำงานอัตโนมัติ ให้เปิด Auto Sync แล้วกด Save Schedule

หมายเหตุ
- โปรแกรมนี้ไม่มีการติดตั้ง Windows Service แล้ว
- ถ้าเปลี่ยนค่า .env หรือ scheduler_settings.json ให้ปิดและเปิดโปรแกรมใหม่
- แนะนำให้เก็บโฟลเดอร์โปรแกรมไว้ตำแหน่งเดิม ไม่ย้ายไปมาระหว่างใช้งาน
